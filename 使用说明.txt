智能模拟交易系统 - Windows 使用说明
====================================

【第一步：安装 Python】
1. 打开 https://www.python.org/downloads/
2. 下载 Python 3.11 或 3.12
3. 安装时一定要勾选 ✅ "Add Python to PATH"
4. 安装完成后重启电脑

【第二步：启动系统】
双击「启动交易系统.bat」即可！
- 首次启动会自动安装依赖，需等待1-2分钟
- 之后每次启动只需几秒钟
- 浏览器会自动打开仪表盘

【功能说明】
- 仪表盘：总资产、盈亏、净值曲线、宏观事件
- 板块热度：6大板块实时情绪分析
- 持仓明细：买入成本、盈亏、止损、财报预警
- 扫描信号：点击「立即扫描」运行完整交易周期
- 每日总结：新闻驱动的板块解读
- 实时新闻：每10分钟自动爬取有价值的财经新闻
- 回测系统：历史策略回测
- 交易记录：所有买卖操作历史

【注意事项】
- 这是模拟交易，不涉及真实资金
- 数据来源：Yahoo Finance（免费）
- 需要网络连接
- 关闭命令行窗口即停止系统
- 数据保存在 C:\Users\你的用户名\.sim_trader.json
